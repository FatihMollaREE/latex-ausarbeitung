Titel:
	Vom RTL zum GDS: Einführung in das VLSI-Design von ICs mit Fokus auf Cadence Innovus
	
Existenzgrund:
	Der Grund für die Existenz der Arbeit von Fatih M. ist das Bedürfnis von Studenten ein grundlegendes Verständnis des Digitalen- Design Flows von integrierten Schaltkreisen zu entwickeln
	
Ziel der Arbeit:
	Ziel ist es Studenten, oder anderen neu Einsteigern mit Basis-Wissen im Bereich von Mikroelektronik einen groben Überblick in das Designen von Digitalen Schaltkreisen zu geben. Besonders wird hierbei ein Fokus auf den groben Flow als auch das Tool Innovus von Cadence gelegt, welches hierfür eine zentrale Rolle spielt
	
Forschungsfragen:
	Wie ist der grobe Ablauf vom RTL Code bis hin zum fertigen Design (GDS file)
	Welche wichtigen Phasen werden hier durchlaufen UND welche Rolle nimmt das Tool Innovus hierbei ein
	
Vorgehensweise:
	Zuerst wird ein grober Überblick über alle Phasen des Design-Flows dargestellt. Anschliessend sollen die wichtigsten Phasen hervorgehoben als auch näher erläutert werden.
	Zuletzt soll auch eine Einführung in die Nutzung des Innovus Tools zur Umsetzung dieser Phasen gegeben werden
